Cette page présente les principales fonctions de la bibliothèque Pandas pour explorer, analyser et visualiser des données.

1. Lire les données stockées dans un fichier csv

pd.read_csv() permet de charger un fichier CSV dans un DataFrame Pandas.


import pandas as pd
df = pd.read_csv('fichier.csv')

Options utiles : sep si le séparateur est un ; sep=';'

2. Vérifier les données

  • df.shape : nombre de lignes et de colonnes
  • df.head(n) : affiche les 5 premières lignes
  • df.tail(n) : affiche les 5 dernières lignes
  • df.info() : résumé des colonnes et types
  • df.describe() : statistiques descriptives des colonnes numériques

3. Obtenir des résultats univariés et bivariés

  • df['colonne'].value_counts() : compte les occurrences de chaque valeur d'une variable qualitative
  • pd.crosstab(df['col1'], df['col2']) : tableau croisé simple de deux variables qualitatives

4. Gestion des valeurs manquantes


df.isnull()
df.notnull()
df.dropna(axis=0, inplace=True)
df.fillna(0, inplace=True)

5. Filtrer les données

  • Retenir une seule colonne : df = df['colonne']
  • Retenir plusieurs colonnes : df = df[['col1','col2']]
  • Retenir par filtrage conditionnel : df=df[df['Age'] > 30]

Autre option pour filtrer: query()

La méthode query() permet de filtrer les lignes d’un DataFrame en utilisant une expression textuelle.

# Exemple : sélectionner les lignes où Age > 30 et Sexe = 'F'
df = df.query("Age > 30 and Sexe == 'F'")

# Exemple avec plusieurs conditions et colonnes
df= df.query("Region == 'Montréal' or Revenu > 50000")

6. Grouper les observations et obtenir des moyennes


df.groupby('Region')['Revenu'].mean()
df.pivot_table(values='Score', index='Classe', columns='Sexe', aggfunc='mean')

7. Fusionner deux bases

  • pd.merge(df1, df2, on='colonne') : fusion sur une clé
  • pd.concat([df1, df2]) : concaténation de DataFrames

8. Visualiser graphiquement


    
import matplotlib.pyplot as plt

t1=df['Age']
t1.plot(kind='hist', bins=10, color='skyblue')
plt.show()

t2=df['Pays'].value_counts()
t2.plot(kind='bar')
plt.show()

t3=df.groupby('Année')['Ventes'].sum()
t3.plot(kind='line')
plt.show()

t4=df['Catégorie'].value_counts()
t4.plot(kind='pie', autopct='%1.1f%%')
plt.show()

9. Exportation des données

  • df.to_csv('fichier.csv', index=False) : export CSV
  • df.to_excel('fichier.xlsx') : export Excel

10. Conseils pratiques

  • Commencer par df.head() et df.info() pour comprendre les données.
  • Utiliser value_counts() et crosstab() pour analyser rapidement les distributions.
  • Les groupby() et pivot_table() sont essentiels pour résumer les données.
  • Visualiser les tendances et anomalies avec plot().
